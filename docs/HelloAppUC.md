## UC2: Personalized Greeting
- **Description:** Display "Hello, <name>!" using command-line arguments.
- **Disadvantage of UC1:** Static output; no user interaction.
- **Preconditions:** Program executed with or without an argument.
- **Main Flow:** 1. Check `args.length`.
  2. If > 0, print `args[0]`.
  3. Else, print "World".
- **Concepts:** Array indexing, String concatenation, Conditional logic (if-else).
